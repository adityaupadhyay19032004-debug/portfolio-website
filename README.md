# Personal Portfolio Website

This project is a modern and responsive personal portfolio website built using HTML, CSS, and JavaScript.

The website showcases my skills, projects, and contact information in a clean and interactive interface. It includes features such as a typing animation, dark/light mode toggle, and dynamic navbar effects.

## Features
- Responsive design for all devices
- Typing text animation
- Dark / Light mode toggle
- Navbar scroll effect
- Smooth and modern UI

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This portfolio website is designed to present my work, skills, and experience as a frontend developer.
